# 🚖 TaxiBot — Telegram Bot на C#

Telegram-бот для служби підтримки таксі з живим чатом між користувачами та операторами.

---

## 📦 Структура проекту

```
TaxiBot/
├── Program.cs                  ← Точка входу, DI-контейнер
├── TaxiBot.csproj
├── Models/
│   └── Models.cs               ← UserSession, OperatorSession, ProblemType
├── Services/
│   ├── BotService.cs           ← Вся логіка бота (BackgroundService)
│   ├── SessionService.cs       ← Зберігання сесій користувачів
│   └── OperatorService.cs      ← Реєстрація та управління операторами
└── Helpers/
    └── KeyboardHelper.cs       ← Inline клавіатури Telegram
```

---

## 🚀 Запуск

### 1. Встановіть .NET 8 SDK
https://dotnet.microsoft.com/download

### 2. Отримайте токен бота
Напишіть [@BotFather](https://t.me/BotFather) у Telegram, створіть бота та скопіюйте токен.

### 3. Встановіть токен

**Windows (CMD):**
```cmd
set TELEGRAM_BOT_TOKEN=ВАШ_ТОКЕН
```

**Linux/macOS:**
```bash
export TELEGRAM_BOT_TOKEN=ВАШ_ТОКЕН
```

Або замініть `"YOUR_BOT_TOKEN_HERE"` прямо в `Program.cs`.

### 4. Запустіть бота
```bash
cd TaxiBot
dotnet run
```

---

## 👨‍💼 Як зареєструватися як оператор

Щоб стати оператором, напишіть боту команду:
```
/operator Костянтин
```

Після цього ви будете отримувати запити від користувачів.

```

──────────              ───                     ────────

"🚗 Аварія"   →   "Опишіть проблему"
"Мав аварію.."→   "Запит прийнято,              ← Отримує контекст запиту
                   очікуйте оператора"
                   "Оператор Костянтин           "Доброго дня, чим можу..."→
                    підключився..."
"Де знаходитися ← ← ← ← ← ← ← ← ← ← ← ← "Де знаходитися машина?"
  машина?"    →  → → → → → → → → → → → → → →  [оператор відповідає]
```

## 📌 Примітки

- Сесії зберігаються **в пам'яті** (перезапуск бота — дані скидаються).
- Для продакшн-версії: додайте базу даних (PostgreSQL / SQLite через EF Core).
- Для масштабування: черга запитів (якщо немає вільного оператора).
